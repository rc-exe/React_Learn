# React Basics — Code & Examples

A side-by-side React learning app: each topic shows **code** on the left and a **live example** on the right.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown (e.g. http://localhost:5173).

## Topics covered

1. **JavaScript Fundamentals** — let/const, arrow functions, destructuring, spread, map/filter/reduce, async/await  
2. **What React Is** — library vs framework, Virtual DOM, state-driven UI  
3. **Components** — functional components, composition  
4. **JSX** — expressions, `className`, self-closing tags  
5. **Props** — parent → child, read-only  
6. **State** — `useState`, functional updates  
7. **Events** — handlers, passing arguments  
8. **Conditional Rendering** — `&&`, ternary, early return  
9. **Lists & Keys** — `map`, `key`  
10. **useEffect** — side effects, dependency array, cleanup  
11. **Controlled Components** — forms, `value` + `onChange`  
12. **Folder Structure** — Vite, organizing components  
13. **Hooks** — useState, useEffect, useRef, useContext  
14. **Lifting State Up** — shared state in parent  
15. **Performance** — re-renders, React.memo  
16. **API Calls** — fetch, loading/error state  
17. **Routing** — react-router-dom, useParams, useNavigate  
18. **Debugging** — errors, DevTools, React DevTools  

Built with **Vite** and **React 18**.
